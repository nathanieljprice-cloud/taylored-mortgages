How to Post a Blog Article

Taylored Mortgages · tayloredmortgages.com

Quick Reference

New articles are added directly through GitHub — no password managers, no CMS login, just a simple web form. From start to published takes about 3 minutes. The site updates automatically within 30 seconds of saving.

1

Go to the posts folder

Open github.com/nathanieljprice-cloud/taylored-mortgages → click the _posts folder.

2

Create a new file

Click Add file → Create new file in the top-right corner of the folder view.

3

Name the file

In the filename box at the top, type: YYYY-MM-DD-short-title.md
Example: 2026-05-08-spring-market-update.md

4

Paste the template and fill it in

Copy this exactly — then replace the values
---
title: "Your Article Title Here"
category: market  # market · tips · refi · loans
date: 2026-05-08
excerpt: "One or two sentences shown on the blog card."
image: https://images.unsplash.com/photo-XXXXXXX?auto=format&fit=crop&w=800&q=75
---

# Write your article here using plain text.
# Press Enter twice for a new paragraph.
# Use ## for section headings.

Your article content starts here. Write naturally —
no special formatting needed for regular paragraphs.

## Example Section Heading

Another paragraph of content goes here.
5

Save (commit)

Scroll to the bottom → click the green Commit changes button. Leave the default message as-is. The site updates in about 30 seconds.

Field reference
title
The article headline. Keep it under 70 characters. Wrap in quotes.
category
One of four options: market, tips, refi, or loans. No quotes, exact spelling.
date
Publish date in YYYY-MM-DD format. Used for sorting newest-first.
excerpt
1–2 sentence preview shown on the blog card. Wrap in quotes.
image
Paste a full Unsplash URL. Find free photos at unsplash.com — copy the image address.
body
Everything after the second ---. Plain paragraphs, ## headings, and bullet lists all work.

Good to know

  • The file name becomes part of the URL — use lowercase letters, numbers, and hyphens only.
  • To find an Unsplash image URL: open the photo on unsplash.com, right-click the image → "Copy image address."
  • New posts appear at the top of the blog grid automatically — no need to edit any other files.
  • To edit a post after publishing: navigate back to the file in _posts/, click the pencil icon, make changes, and commit.
  • To unpublish: rename the file to start with an underscore (e.g. _draft-title.md) and commit.
